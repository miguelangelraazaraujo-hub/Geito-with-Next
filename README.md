## Sobre el Proyecto

**Geito with Next** es un proyecto desarrollado con Next.js que busca ofrecer una base moderna y eficiente para aplicaciones web full stack.

Características principales:

* Arquitectura moderna basada en Next.js
* Renderizado híbrido (SSR / SSG)
* Organización escalable del código
* Preparado para despliegues rápidos
* Optimización de rendimiento y SEO
* Integración sencilla con APIs y servicios externos

Este repositorio sirve como base para construir aplicaciones web modernas utilizando el ecosistema React + Next.js.

## Comenzando

### Prerrequisitos

Asegúrate de tener instalado:

* Node.js 18+
* npm, yarn o pnpm
* Git

---

### Instalación

1. Clona el repositorio

```bash
git clone https://github.com/miguelangelraazaraujo-hub/Geito-with-Next.git
```

2. Entra en el directorio del proyecto

```bash
cd Geito-with-Next
```

3. Instala las dependencias

```bash
npm install
```

o

```bash
yarn install
```

4. Ejecuta el servidor de desarrollo

```bash
npm run dev
```

5. Abre tu navegador en:

```txt
http://localhost:3000
```
